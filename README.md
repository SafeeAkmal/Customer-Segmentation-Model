# Customer Segmentation Using Clustering Algorithms

> Data Mining & Machine Learning | February 2026  
> Muhammad Junaid Minhas (23L-2559) & Safee Akmal (23L-2556)

## Problem Statement

This project applies unsupervised machine learning to the **Customer Personality
Analysis** dataset (Kaggle, ~2,240 customers, 29 features) to identify distinct
customer segments using K-Means, Hierarchical Clustering, and DBSCAN.
